サブスクマネージャー 仕様書


============================================================
概要
============================================================

アプリ名:   サブスクマネージャー
ジャンル:   サブスクリプション管理ツール

どんなアプリ?
  契約中のサブスクリプションを一覧管理するアプリ。
  月額/年額の合計費用を自動計算し、更新日をカレンダーで可視化。
  カテゴリ別の支出割合や月別推移をグラフで確認できる。
  「自分が毎月いくら払っているか」を一目で把握して無駄を発見する。

誰が使う?
  サブスクが増えて管理しきれない人、毎月の固定費を見直したい人。


============================================================
機能一覧
============================================================

サブスク登録/編集/削除
  サービス名
  カテゴリ (動画 / 音楽 / ゲーム / ツール / 学習 / 通信 / ニュース / その他)
  金額 (数値入力)
  請求サイクル (月額 / 年額)
  次回更新日 (日付入力)
  メモ (自由記述)
  ステータス (契約中 / 解約済み)

サブスク一覧 (メイン画面)
  カード形式で一覧表示 (カテゴリカラー + サービス名 + 月額換算金額)
  カテゴリ別フィルター (すべて + 8カテゴリ)
  ステータスフィルター (契約中 / 解約済み / すべて)
  サービス名での検索
  空の時は登録誘導メッセージ

ダッシュボード
  月額合計 (全サブスクを月額換算して合計)
  年額換算合計
  契約中のサービス数
  今月更新予定のサービス数

更新カレンダー
  月間カレンダーに更新日をマーク表示
  更新日が近い (7日以内) サブスクを警告カラーでハイライト
  前月/翌月ナビゲーション

統計
  カテゴリ別支出割合 (円グラフ、Canvas)
  月別支出推移 (棒グラフ、Canvas)

テーマ
  ダークモード / ライトモード切り替え


============================================================
画面構成
============================================================

ヘッダー
  ロゴ + アプリ名
  ナビゲーション (一覧 / カレンダー / 統計)
  テーマ切替ボタン

一覧画面 (メイン)
  ダッシュボードカード4枚 (月額合計 / 年額 / 契約数 / 今月更新)
  カテゴリタブ (すべて + 8カテゴリ)
  ステータス切替 (契約中 / 解約済み / すべて)
  検索バー
  サブスクカードグリッド
  右下にFABボタン (サブスク追加)

カレンダー画面
  月表示カレンダー (7列グリッド)
  各日に更新があるサブスク名を表示
  7日以内の更新は警告カラー

統計画面
  カテゴリ別支出円グラフ
  月別支出推移棒グラフ

サブスク登録/編集モーダル
  サービス名 / カテゴリ (セレクト) / 金額 / 請求サイクル (セレクト)
  次回更新日 (date input)
  ステータス (セレクト)
  メモ (テキストエリア)
  保存 / キャンセル
